Test Browser Extension

This browser extension is a powerful tool that helps you analyze web pages and manage notifications. It provides quick access to page statistics and allows you to display custom notifications while browsing.

How to Install the Extension

  1. Open your Chrome browser (or other Chromium-based browser)
  2. Navigate to chrome://extensions in your browser's address bar ?Browser-specific extension pages:
    • Chrome: chrome://extensions
    • Edge: edge://extensions
    • Brave: brave://extensions
    • Vivaldi: vivaldi://extensions
  3. Enable "Developer mode" by clicking the toggle switch in the top-right corner
  4. Click the "Load unpacked" button that appears
  5. Select the folder containing this extension's files
  6. The extension should now appear in your browser toolbar
Note: After installation, you may need to pin the extension to your toolbar to easily access its features.

Features

  • Analyze web pages to view statistics about their structure
  • Display custom notifications on any webpage
  • Customize the extension's appearance and behavior through the options page
  • Built using modern Chrome extension technologies (Manifest V3)
